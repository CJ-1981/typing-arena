# ⚡ TypeClash: Real-time Multi-Language Typing Arena

A lightning-fast, peer-to-peer multiplayer typing game built for speed and endurance. TypeClash allows you to practice your typing skills solo or challenge friends in real-time without the need for a dedicated backend server, leveraging the power of WebRTC.

## 🌟 Features

- **Peer-to-Peer Multiplayer**: Challenge friends directly with secure, low-latency WebRTC (powered by PeerJS). No server needed!
- **Solo Practice Arena**: Hone your skills offline or while waiting for a match.
- **Multi-Language Support**:
  - English (Standard Quotes)
  - Deutsch (German Umlauts ä,ö,ü,ß)
  - 한국어 (Korean / Jamo Engine)
- **Customizable Difficulty**: Choose between short phrases for fast reflexes, medium quotes, or long endurance sentences.
- **Flash Typing Mode**: Strip all casing and punctuation for pure, raw speed testing.
- **Personalized Profiles**: Customize your nickname and choose from a variety of avatar theme colors.
- **Audio Effects**: Built-in synth audio feedback for a more immersive typing experience.

## 🛠️ Technology Stack

- **HTML5/CSS3/JavaScript**
- **Tailwind CSS**: For a sleek, modern, neon-inspired UI.
- **PeerJS**: Serverless WebRTC engine.
- **Lucide Icons**: Clean, consistent iconography.
- **Google Fonts**: Plus Jakarta Sans & JetBrains Mono.

## 🚀 Getting Started

Simply open `index.html` in your web browser. 

- To play solo, click on **Solo Practice Arena**.
- To play with a friend, configure your profile and create a host lobby, then share your connection ID!

Enjoy the arena!
